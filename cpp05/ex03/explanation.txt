# Exercise 03: Interns - Factory Pattern & Function Pointers

## 1. Design Pattern: Factory (Lmasna3)
Htta daba, konna kancreeyi forms b yeddina: `new RobotomyRequestForm(...)`.
Walakin f programe kbir, user ghadi yktb "robotomy request" f clavier, machi code.
Khassna chi haja kat-t7awel **String** ("robotomy request") -> **Object** (`new Robotomy...`).
Hada howa dawr dial class `Intern`. Hiya "Usine" (Factory) katntaj l forms.

## 2. El Mosiba d `If / Else If`
L hal sahel (o lkassoul) howa:
```cpp
if (name == "robotomy") return new Robotomy...
else if (name == "shrubbery") return new Shrubbery...
else if (name == "presidential") return new Presidential...
else error...
```
Hada smito "Spaghetti Code". Ila 3ndna 100 type d forms, atwalli 3ndna 200 stor d if/else. O slower f l execution (O(N)).

## 3. L Hal Nqi: Array of Function Pointers
C++ (w C) kaykhllik tkhzen "address" dial function f variable.
Ach darna f `Intern.cpp`?

1.  **Derna static helper functions**:
    Functions sghar bhal `makeRobotomy` li katrejje3 `new RobotomyRequestForm(target)`.
    Hado houma "l 3mmal" sghar.

2.  **Derna Array of Pointers**:
    ```cpp
    AForm* (*forms[])(std::string target) = {
        &makeShrubbery,
        &makeRobotomy,
        &makePresidential
    };
    ```
    -   `AForm*`: Type de retour.
    -   `(*forms[])`: Array smitou `forms`.
    -   `(std::string target)`: Arguments li kayakhdo douk functions.
    -   `{ ... }`: L qaima dial addresses d functions.

3.  **Derna Array of Names**:
    `string formNames[] = { "shrubbery creation", ... };`

4.  **Loop**:
    Kanqlbo 3la `name` f `formNames`. Ila lqinah f index `i`, kan3iyto l `forms[i](target)`.
    Salina! Code nqi, sahel tzid fih form jdida (zidha f array safi).

## 4. Return Type & Abstract Classes
L function `makeForm` katrejje3 `AForm*`.
Wakha hna kancreeyiw `RobotomyRequestForm` (li hiya bent), kanrej3oha 3la chakl `AForm*` (L mère).
Hada howa **Polymorphism** li dwin 3lih f ex02.
L user (main.cpp) mayhmouch wash robot ola chjra, kayhemmo anna 3aydo `AForm` yqder ysinyiha w yexecutiha.

## 5. Memory Management (Attention!)
`Intern` katrejje3 `new` pointer.
Ya3ni hadak li 3iyet l `makeForm` (ghaliban `main.cpp`) khasso ydir `delete` melli ysali.
Ila nssit `delete`, 3ndk Memory Leak.
L intern ghir "khalqat" l form w 3tatha lik. Masoulya d nettoyage ntaqlet lik.
